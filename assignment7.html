A7: Final Project
For my final project I plan to design and print ring splints. Ring splints are used to keep finger joints properly aligned to improve function and reduce pain. I need them for tasks like writing with a pen. There's mainly two options for ring splints - cheap ugly ones, or extremely expensive silver ones. So I started printing ring splints soon after getting my printer using a design from thingiverse. But they don't quiet fit right. So I want to adjust it to create a better fit, and work off the design to add decorative elements to create unique styles.

I will continue to work off of this STL file from thingiverse (and this thicker variation) to create a new design.

This will involve CAD modeling, printing, and finishing (sanding and polishing)

Materials: PLA filament, sand paper in grits 120-5000 (already owned)

Timeline:
1. Finalize adjusted fit for multiple joints and fingers
2. Work on design changes and embellishments
3. Print
4. Sand and polish

Also depending on time I'm interested in created necklace pendants that match the rings.
